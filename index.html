Homework #5: Custom Visualization Design | Daniel Osawaru - dosawaru@asu.edu

This visualization presents a force-directed network of the top 100 Instagram influencers base on Kaggle Dataset. Each node represents a influencer, using their profile picture, and the size of each node reflects their influence score. Category hubs are connected to the influencer they represent. You can hover over the nodes to see details about the influencers, click a node to visit their Instagram page, and drag the nodes to explore the network layout. There is also a subtle animatioin that keeps the nodes gently moving sround the screen.
Marks: Circles (Nodes), Lines (Edges), Text (Labels) Channels: Size (Influence Scores), Color (Differnet Categories), Shape (Influencers and Category Hubs) Visualization Type: Combinatorial Interaction: Hover (Details), Click (Visit Instagram), Drag (Explore Network), Animation (Subtle Movement) Data: Top 100 Instagram Influencers (Note: The dataset was modified to include categories that best describes each influencer, and a Python script was used to scrape the profile pictures of the influencers from their Instagram page)
